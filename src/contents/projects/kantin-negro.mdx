## Case Study

Kantin sekolah berencana membuat sistem pemesanan makanan dan minuman secara online. Sistem yang akan dikembangkan terdiri atas dua pengguna yaitu siswa dan admin stan. Aplikasi ini hanya digunakan untuk melakukan pemesanan online, tidak mencakup fitur pembayaran.

---

## Goals

### User

- Siswa dapat register user sebagai pelanggan.
- Siswa dapat dapat login ke halaman pemesanan.
- Siswa dapat melihat menu makanan dan minuman.
- Siswa dapat memesan makanan dan minuman dengan diskon sesuai momen tertentu.
- Siswa dapat melihat status pesanan makanannya.
- Siswa dapat melihat histori transaksi berdasarkan bulan.
- Siswa dapat mencetak buk/nota pemesanan.

### Admin

- Calon pemilik stan dapat register untuk mendaftarkan stannya dan sebagai admin stan.
- Admin stan dapat login ke halaman pengelolaan stan.
- Admin stan dapat update data profil stan.
- Admin stan dapat CRUD data pelanggan.
- Admin stan dapat CRUD data makanan dan minuman.
- Admin stan dapat CRUD data diskon dan memilih makanan dan minuman yang didiskon.
- Admin stan dapat mengkonfirmasi dan mengubah status pesanannya.
- Admin stan dapat melihat semua data pemesanan berdasarkan bulan.
- Admin stan dapat melihat rekap pemasukan perbulan.
